# **App Name**: PDFusion

## Core Features:

- File Upload: Allow users to upload PDF files via drag and drop or manual selection.
- File Reordering: Enable users to reorder the uploaded PDF files.
- File Exclusion: Provide an option for users to exclude specific PDF files from the merge.
- PDF Merging: Merge the selected and ordered PDF files into a single PDF document via API Gateway and Lambda.

## Style Guidelines:

- Primary color: A vibrant blue (#F20528) to convey trust and efficiency, inspired by the precision needed in document management.
- Background color: A light gray (#EDB4C0) provides a clean and unobtrusive backdrop, same hue as primary, but highly desaturated.
- Accent color: A teal color (#EBEEF2), analogous to the blue but more saturated and less bright, to highlight key actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its modern, neutral look suitable for headlines and body text
- Use simple, intuitive icons for file actions like upload, reorder, and merge.
- A clean and intuitive drag-and-drop interface for easy file management.
- Subtle animations to provide feedback on file actions, like upload and merge completion.
