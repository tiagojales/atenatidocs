# AtenaDocs - Online PDF Tools

This is a Next.js project for AtenaDocs, a web application for merging PDF files.

The frontend is built with Next.js and ShadCN UI, and the backend is a serverless Python function running on AWS Lambda.

## Getting Started

To get started with local development, take a look at the instructions in `docs/DOCUMENTATION.md`.

The main application page is located at `src/app/page.tsx`.
